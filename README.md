# 🚀 FactuFly — Landing Page & Sistema de Facturación Electrónica SUNAT

> **FactuFly** es el punto de venta (POS) y sistema de facturación electrónica todo en uno para negocios peruanos. 100% homologado ante la SUNAT (PSE), diseñado para automatizar ventas, cobros, inventario y emisión de comprobantes electrónicos en segundos.

🌐 **Sistema en Producción**: [https://factufly.ideatec.com.pe/](https://factufly.ideatec.com.pe/)

---

## ✨ Características Principales

- 🧾 **Comprobantes Electrónicos SUNAT**: Emisión instantánea de Boletas de Venta, Facturas Electrónicas, Notas de Crédito, Notas de Débito y Guías de Remisión Remitente (GRE).
- 🛒 **Punto de Venta (POS) & Inventario**: Control de stock en tiempo real, Kardex automático y lectura rápida por código de barras.
- 🔍 **Consultas RENIEC y SUNAT**: Validación automatizada de DNI y RUC al instante.
- 🖨️ **Impresión Térmica**: Formato de impresión adaptable para ticketeras de 58mm y 80mm.
- ⚡ **API para Desarrolladores**: Documentación interactiva e integración directa (`/apifactufly`).
- 🌙 **Diseño Premium & Modo Oscuro**: Interfaz receptiva de alta gama con soporte nativo para Light Mode y Dark Mode.

## 🛠️ Stack Tecnológico

- **Frontend**: React 19 + TypeScript

- **Estilos**: Tailwind CSS v4

- **Animaciones**: Motion (Framer Motion)

## ⚙️ Instalación y Ejecución Local

### Prerrequisitos
- [Node.js](https://nodejs.org/) (v18.0.0 o superior recomendada)
- `npm` instalado

### Instrucciones

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/CODLuisP/ladingFactuFly.git
   cd landingFactuFly
   ```

2. **Instalar dependencias:**
   ```bash
   npm install
   ```

3. **Iniciar en modo desarrollo:**
   ```bash
   npm run dev
   ```
   La landing se ejecutará en: `http://localhost:3001`

4. **Verificación de tipos (TypeScript):**
   ```bash
   npm run lint
   ```

5. **Compilar para producción:**
   ```bash
   npm run build
   ```
